# Markdown 기본 사용법 (MD → MD 설명)

> 이 문서는 **Markdown으로 Markdown을 설명**하는 팀 노트용 기본 가이드입니다.

---

## 📌 Markdown이란?

* 가볍게 문서를 작성하는 문법
* HTML보다 **간단하고 읽기 쉬움**
* GitHub, GitLab, Notion 등에서 널리 사용

👉 팀 노트, 기술 문서, README에 최적

---

## 🧱 제목 (Headings)

```md
# 제목 1 (문서 제목)
## 제목 2 (큰 섹션)
### 제목 3 (세부 섹션)
```

### 사용 예

```md
# Docker 기초
## 설치
### macOS
```

---

## ✍️ 문단과 줄바꿈

```md
이것은 한 문단입니다.

빈 줄을 넣으면 새 문단이 됩니다.
```

## 📋 목록 (Lists)

### 순서 없는 목록

```md

- MySQL
- Docker
```

### 순서 있는 목록

```md
1. 설치
2. 설정
3. 실행
```

---

## ✅ 체크리스트 (팀 작업용)

```md
- [ ] PHP 설치
- [x] Docker 실행
- [ ] DB 연결
```

## 📁 팀 노트 작성 규칙 (요약)

* 파일명: `kebab-case.md`
* 제목은 `#`부터 시작
* 명령어는 **항상 코드 블록**
* 한 섹션 = 한 주제
